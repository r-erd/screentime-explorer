Full Disk Access Required

Screen Time needs Full Disk Access to read Apple's usage database on your Mac.

  1. Open System Settings
  2. Go to Privacy & Security → Full Disk Access
  3. Enable access for Screen Time

iPhone & iPad Data
Only Mac usage is available on macOS 13+.
Apple moved synced device Screen Time data into a private, sandboxed database (RMAdminStore-Cloud.sqlite) that is protected beyond Full Disk Access — only Apple's own ScreenTimeAgent process can read it. This is an Apple limitation that cannot be worked around.

Colors in charts still distinguish between devices if data is present: Mac   iPhone   iPad
